# LangSmith

## Introduction

**LangSmith** is a platform for building production-grade LLM applications.

It lets you debug, test, evaluate, and monitor LLM powered applications built on **any framework or language** and seamlessly integrates with **LangChain**, the go-to open source framework for building with LLMs.

:::tip Not Using LangChain?
LangSmith is a standalone service with no dependency on LangChain. If you're not using LangChain, you can still tap into everything LangSmith has to offer.
:::
