# Self-hosting LangSmith with Docker

This guide will walk you through the process of setting up LangSmith on your own infrastructure using Docker.

:::important Enterprise License Required
Self-hosting LangSmith requires an enterprise license. Please contact us at <a href="mailto:support@langchain.dev">support@langchain.dev</a> for more information.
:::
:::important Migrating from LangSmith 0.1.0 to 0.2.0
LangSmith 0.2.0 introduces a new dependency on Clickhouse for run storage. If you wish to retain runs in LangSmith from versions of LangSmith prior to 0.2.0, you will need to complete a migration process. You can view the upgrade guide here. If you need assistance, please reach out to support@langchain.dev.
:::
